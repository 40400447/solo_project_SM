# solo_project_SM
Additional materials for project- Is 12 Yards the Optimal Penalty Distance? A Mathematical Model of Penalty Kicks.

Contents
1. Figure 2 code- code used to produce the graph of time taken for a penalty kick to reach the goal line as a function of the distance of the penalty spot.
2. Figure 3 code- code used to produce the graph of goalkeeper reachable distance as a function of penalty spot distance.
3. Figure 4 code- code used to produce simulated scoring probability versus penalty distance from Monte Carlo sampling of random shot locations.
4. Figure 6 code- code used to produce the graph of the probability of scoring using uniform shot placement compared with real-world placement data.
5. Anticipation figure- figure showing the probability of the goalkeeper correctly guessing what side the ball will go to as a function of the distance of the penalty spot.
6. Anticipation figure code- code used to produce Anticipation Figure.
